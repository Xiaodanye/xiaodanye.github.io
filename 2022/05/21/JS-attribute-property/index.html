attribute & property

  1. attribute 是 DOM 元素在 Html 标签上拥有的特性
    • 大小写不敏感
    • 值永远是字符串或 null
    • 能够自定义特性 data-*
    • 从语义上，更倾向于不可变更的（例如href）
  2. property 是 DOM 元素在 JavaScript 中作为对象拥有的属性
    • 大小写敏感
    • 值可以是任意合法 js 类型
    • 无法获取自定义属性 data-*
    • 从语义上，更倾向于在其生命周期中是可变的

PS：对于 html 的标准属性来说，attribute 和 property 是同步的，是会自动更新的。（input例外）
